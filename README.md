# LINE-MAN Search Ranking Model

Machine learning model predicting restaurant click probability using logistic regression.

## Problem
Improve restaurant ranking in keyword-based search results to increase click-through rate (CTR).

## Dataset
- Impression Log
- Click Log
- User Metadata
- Restaurant Metadata

## Technologies
- Python
- Pandas
- NumPy
- Scikit-learn
- Jupyter Notebook

## Approach
1. Data cleaning and preprocessing
2. Exploratory data analysis
3. Feature engineering
4. Logistic regression model training
5. Model evaluation

## Result
The model predicts the probability that a restaurant will be clicked and can be used to rank search results.
